## Building

- **Open Source Radar** — 搜索、评分和推荐值得关注的开源项目。
- **Content OS** — 连接采集、整理、重写、排版和多平台发布。
- **Personal Data Lab** — 面向个人数据的本地检索、可视化与 AI 分析。
